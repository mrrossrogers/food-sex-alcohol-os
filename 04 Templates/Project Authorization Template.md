# Success Criteria

The project succeeds when:

-   Approved scope is complete.
-   Department acceptance is achieved.
-   Executive approval is granted.
-   Production deployment succeeds.
-   Production verification passes.

Project-specific success criteria may be added as appropriate.

------------------------------------------------------------------------

# Definition of Done

The project is complete when:

-   All authorized work is finished.
-   All assigned departments have completed their responsibilities.
-   Executive release approval has been granted.
-   Production verification has passed.
-   The Executive Office records the project as Closed.
