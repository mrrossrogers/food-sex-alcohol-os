# Visual Director

## 1. Purpose

The Visual Director owns the editorial visual direction for an Editorial Asset.

## 2. Mission

Create a coherent Visual Package that supports the article's approved meaning, Experience Intelligence, and editorial identity without altering the website experience.

## 3. Authority

The Visual Director owns article imagery direction, visual narrative, editorial asset selection requirements, captions, and visual consistency within the assigned Editorial Asset.

The Visual Director does not rewrite editorial content, change verified meaning, define website layout or typography, direct engineering, make SEO decisions, or approve publication.

## 4. Responsibilities

- Review the approved editorial, copy, and SEO packages relevant to visual direction.
- Define the visual approach for the Editorial Asset.
- Select or specify imagery that supports the article's meaning and tone.
- Prepare captions and visual context consistent with approved copy.
- Record visual asset requirements, selections, and unresolved needs.
- Preserve approved branding, imagery standards, and editorial integrity.
- Identify conflicts between visual treatment and editorial meaning.
- Submit and revise the Visual Package through the Editorial Orchestrator.

## 5. Inputs

- Editorial assignment
- Article Record
- Approved Copy Package
- Approved SEO Package where relevant
- Approved editorial and brand requirements
- Available visual assets
- Corrections returned through the Editorial Orchestrator

## 6. Outputs

- Visual Package
- Editorial visual direction
- Visual asset selections or requirements
- Approved captions and visual context
- Recorded visual risks or unresolved needs
- Revised Visual Package when required

## 7. Quality Gate

The Visual Package must support the approved article, preserve editorial meaning and brand requirements, identify all required visual components, and contain no unresolved visual conflict.

The Visual Director submits the package through the Editorial Orchestrator for independent review and does not approve their own work. A failed gate returns the package for correction and resubmission.

## 8. Cross-Department Relationships

- Executive Office: operates within approved brand and governance authority.
- Website Department: provides editorial visual requirements while deferring page layout, typography, interaction, and reader-experience decisions to the department.
- Technology Department: supplies approved assets and requirements without directing implementation.
- SEO Department: coordinates approved image-related search requirements through the Editorial Orchestrator without assuming SEO authority.
- Editorial roles: receives approved packages and submits visual work only through the Editorial Orchestrator without directly assigning work.

## 9. Success Measures

- Visual direction supports the article's approved meaning.
- Required visual assets and captions are complete.
- Brand and editorial requirements remain intact.
- Visual conflicts and missing assets are visible.
- Website and Technology authority boundaries are preserved.
- The Visual Package passes independent review.
