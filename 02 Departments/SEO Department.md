# Purpose

The SEO Department is responsible for the search discoverability of Food
Sex Alcohol.

The department ensures that approved content and website experiences can
be discovered, crawled, understood, and indexed by search engines.

It does not own editorial conclusions. It does not own reader-experience
design. It does not perform engineering implementation. It does not own
Executive governance.

------------------------------------------------------------------------

# Mission

Maximize the discoverability of Food Sex Alcohol without compromising
editorial integrity, reader trust, brand quality, or the approved
website experience.

------------------------------------------------------------------------

# Department Authority

The SEO Department may:

-   Define technical SEO requirements.
-   Evaluate crawlability.
-   Evaluate indexability.
-   Validate canonical URLs.
-   Review URL structure.
-   Review internal linking.
-   Define metadata requirements.
-   Define structured-data requirements.
-   Validate sitemap and robots behavior.
-   Review search-engine risks.
-   Recommend search improvements.
-   Accept or reject SEO conformance within assigned projects.

------------------------------------------------------------------------

# The SEO Department shall not:

-   Rewrite editorial conclusions.
-   Determine editorial opinions.
-   Redesign the website.
-   Write production code.
-   Modify repositories.
-   Approve production releases.
-   Change Executive priorities.
-   Expand approved project scope.
-   Prioritize company projects.
-   Assign work to another department.

------------------------------------------------------------------------

# Responsibilities

The SEO Department owns:

-   Crawlability
-   Indexability
-   Canonical validation
-   Search-friendly URL standards
-   XML sitemap requirements
-   robots.txt requirements
-   Structured-data requirements
-   Metadata standards
-   Internal-linking validation
-   Search Console validation
-   Bing Webmaster Tools validation
-   Search-performance recommendations
-   Technical SEO acceptance

------------------------------------------------------------------------

# Validation Authority

The SEO Department performs independent validation when search behavior
is materially affected.

Validation classifications:

-   Accept
-   Accept with Conditions
-   Reject

SEO validation does not authorize commit, merge, deployment, production
release, new projects, or editorial rewriting.

------------------------------------------------------------------------

# Relationship with Other Departments

Executive Office: company priorities, project authorization, risk
acceptance, release approval, project closure.

Website: reader experience, information architecture, navigation,
website acceptance.

Technology: implementation, code quality, builds, deployment readiness,
production verification.

Editorial: story accuracy, editorial voice, conclusions, publishing
decisions.

Analytics: measurement, reporting, performance interpretation.

------------------------------------------------------------------------

# Typical Responsibilities During Projects

-   Review approved architecture for search implications.
-   Define canonical behavior.
-   Validate permanent URLs.
-   Confirm crawl paths.
-   Identify duplicate URL risks.
-   Review metadata and structured data.
-   Validate production search behavior.
-   Report search risks without expanding scope.

------------------------------------------------------------------------

# Guiding Principles

-   Protect editorial integrity.
-   Protect reader trust.
-   Prefer durable search value.
-   Separate defects from future improvements.
-   Base recommendations on evidence.
-   Respect project scope.

------------------------------------------------------------------------

# Success Standard

Food Sex Alcohol should be discoverable, technically understandable, and
indexable while remaining a premium editorial publication created for
readers first.
